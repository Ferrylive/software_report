## 📝 修改方法

### 坐标系统说明

- **x坐标**：从左到右，范围大约 0-1100
- **y坐标**：从上到下，范围大约 0-600
- **上颌**：y值较大（靠近底部）
- **下颌**：y值较小（靠近顶部）

### 修改示例

假设你想把 **18号牙**（右上第三磨牙）向右移动20像素，向下移动10像素：

**修改前**：
```javascript
{x: 130, y: 540},  // 18号牙
```

**修改后**：
```javascript
{x: 150, y: 550},  // 18号牙（x+20, y+10）
```

### 注意事项

1. **保持对称**：左右对称的牙齿（如11和21）应该保持x坐标对称
2. **保持顺序**：数组中的顺序必须与 `upperTeeth` 和 `lowerTeeth` 数组对应
3. **两个地方都要改**：牙齿分布图和治疗方案图的位置要一致
