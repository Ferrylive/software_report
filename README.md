# 口腔健康评估报告 - Version 3

渡生科技 - 渡芽AI智能口腔健康评估系统报告

## 📋 项目说明

这是一个基于HTML的口腔健康评估报告系统，支持：
- 响应式设计，适配桌面和移动设备
- 交互式牙齿分布图
- 点击牙齿查看详细信息
- 完整的健康评估和治疗建议

## 📂 文件结构

```
version3/
├── report.html          # 主报告文件（在浏览器中打开此文件）
├── result.json          # 报告数据（可选，report.html中包含示例数据）
├── report.pdf           # PDF格式报告
├── images/              # 图片资源
│   ├── Processed image/ # 牙齿处理后的PNG图片
│   └── *.png            # 原始照片
├── scripts/             # 报告生成脚本
├── samples/             # 示例文件
└── README.md            # 本文件
```

## 🚀 使用方法

### 本地查看

直接双击 `report.html` 文件，在浏览器中打开即可查看。

### 局域网访问（手机访问）

1. 打开命令行，进入 `version3` 目录
2. 运行 Python HTTP 服务器：
   ```bash
   python -m http.server 8000
   ```
3. 在手机浏览器中访问：
   - `http://[电脑IP]:8000/report.html`
   - 例如：`http://10.19.125.118:8000/report.html`

**注意：** 确保手机和电脑在同一Wi-Fi网络下，且防火墙允许端口8000访问。

### 在线部署

#### Netlify Drop（推荐）

1. 访问 https://app.netlify.com/drop
2. 将整个 `version3` 文件夹拖拽上传
3. 获取部署链接

#### GitHub Pages

1. 创建GitHub仓库
2. 上传文件到仓库
3. 在 Settings > Pages 中启用 GitHub Pages
4. 访问 `https://你的用户名.github.io/仓库名/`

#### Vercel

1. 访问 https://vercel.com
2. 使用GitHub登录并导入仓库
3. 自动部署完成

## 🔧 技术特性

- 纯HTML/CSS/JavaScript，无需后端服务器
- 响应式设计，完美适配手机和电脑
- 交互式牙齿分布图，支持点击查看详情
- 自动数据加载：优先使用 `result.json`，失败时使用嵌入示例数据
- 支持PDF导出（使用浏览器打印功能）

## 📱 功能说明

### 牙齿问题分布图

- 点击任意牙齿可查看详细信息
- 健康牙齿显示为绿色
- 单个问题牙齿显示为黄色
- 多个问题牙齿显示为橙色

### 详情页面包含

- 牙齿编号和位置信息
- 检测图像（如果有）
- 问题分析和可能原因
- 预防与护理建议
- 治疗建议

## ❓ 常见问题

**Q: 图片无法显示？**  
A: 确保 `images` 文件夹与 `report.html` 在同一目录下，且路径正确。

**Q: 数据无法加载？**  
A: 报告会自动使用嵌入的示例数据，不影响查看。

**Q: 手机端显示异常？**  
A: 清除浏览器缓存，或使用无痕模式打开。

**Q: 局域网无法访问？**  
A: 检查防火墙设置，确保允许端口8000的访问。
